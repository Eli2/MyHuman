MyHumanoid
==========

MyHumanoid is a modernized fork of MakeHuman 0.9.1

Install dependencies under Debian and derivatives e.g. Ubuntu

~~~~
apt install libsdl2-dev libglew-dev libglm-dev libphysfs-dev libgtk-3-dev
~~~~
